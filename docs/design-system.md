# Design System

## Theme

"Neubrutalist" (Whalies inspired). Defined in `assets/styles.css`.

## Rules

- **Borders:** `border-4 border-whalies-navy` for containers.
- **Radius:** `rounded-4xl` (`--radius-4xl: 2rem`) for containers;
  `rounded-button` (`--radius-button: 1.5rem`) for buttons.
- **Shadows:** `shadow-cartoon` (hard offset shadow, defined as
  `--shadow-cartoon`). Hover state: `shadow-cartoon-hover`.

## Colors

Use custom CSS variables / utility classes:

- `bg-pastel-pink`
- `bg-pastel-yellow`
- `bg-pastel-blue`
- `bg-pastel-purple`
- `bg-pastel-mint`

## Typography

Google Fonts: **Fredoka** (cartoon/headings, via `--font-cartoon`), **Nunito**
(body, via `--font-body`).
